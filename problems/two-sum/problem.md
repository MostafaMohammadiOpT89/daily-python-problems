Given an array of integers `nums` and an integer `target`, return the indices of the two numbers such that they add up to `target`.

You may assume that exactly one valid pair exists, and you may not use the same element twice.

## Function
Implement `solve()` so it reads input and prints the answer.

### Input format
- First line: integer `n`, the number of elements
- Second line: `n` space-separated integers `nums`
- Third line: integer `target`

### Output format
- Print two indices in increasing order, separated by a space

### Examples
Input:
`4\n2 7 11 15\n9`

Output:
`0 1`

Input:
`3\n3 2 4\n6`

Output:
`1 2`
